決定性有限オートマトン (DFA) は、次の 5 項組  <br/>
A = (Q, Σ, δ, q0, F)  <br/>
 Q：状態の有限集合  <br/>
 Σ：有限のアルファベット (入力記号の有限集合)  <br/>
 δ：遷移関数 (q, a) 7→ p  <br/>
 q0 ∈ Q：開始状態  <br/>
 F ⊆ Q：最終状態 (あるいは、受理状態) の集合  <br/>

非決定性有限オートマトン (NFA) の定義  <br/>
A = (Q, Σ, δ, q0, F)  <br/>
 Q：状態の有限集合  <br/>
 Σ：有限のアルファベット  <br/>
 δ：遷移関数 (q, a) 7→ {p1, · · · , pn}  <br/>
 q0 ∈ Q：開始状態  <br/>
 F ⊆ Q：最終状態 (あるいは、受理状態) の集合  <br/>

G:  S -> A | a | BB, A -> b, C -> c | ABa <br/>
